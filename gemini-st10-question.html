ST-10 자가테스트 결과 화면 - Gemini 질문용

Figma 디자인을 Android Compose로 구현하다 실패한 작업입니다. 아래 내용을 복사해서 Gemini에게 질문하세요.

1. Figma 디자인 정보

URL: https://www.figma.com/design/jTxTaPrc0c2cyGeSRknNEN/cole?node-id=317-3002

화면명: ST-10 자가테스트 결과 화면 (data-name="ST-10")

2. Figma 디자인 스펙 (구성 요소)

  • 상단 제목: "장원영 님의" + "스마트폰 중독 테스트" (보라색) + "결과입니다"
  • 반원형 게이지:
    • 250~500 점수 스케일, 라벨 (250, 350, 500)
    • 아크 그라데이션: 초록→파랑→노랑→주황/빨강
    • 중앙에 큰 점수 (예: 430)
    • 삼각형 인디케이터가 아크 상의 점수 위치를 가리킴
  • 메시지: "스마트폰 사용 습관이 건강해요!" (점수 구간별 다름)
  • 습관 항목 3개: 각 Row에 [라벨(좌) | 아이콘(초록/빨강/노랑) + 상태텍스트(우)]
    • 수면장애 - icon-Normal(초록) - 정상
    • 집중력 저하 - icon-Danger(빨강) - 정상
    • 사용시간 증가 - icon-Caution(노랑) - 주의
  • 디스클레임: "본 자가진단 테스트는 스마트폰 사용 습관을 점검하기 위한 참고 자료이며, 의학적 진단 도구가 아님을 알려드립니다"
  • 버튼: "계속 진행" (Primary), "돌아가기" (Ghost)

3. 현재 구현 vs Figma 차이 (실패 원인)

요소Figma 스펙현재 구현
게이지 반원형 아크, 250~500 스케일, 그라데이션, 삼각형 인디케이터 가로 막대(260x12dp), resultType별 2색 그라데이션, 화살표
제목 구조 "OO 님의 스마트폰 중독 테스트 결과입니다" (한 블록) "OO 님은" + 메인문구 + 부가문구 (분리된 구조)
점수 표시 게이지 중앙에 큰 점수(430) + 아크 위 인디케이터 막대 위 화살표만, 점수 숫자 없음
습관 아이콘 12px 원형 아이콘 (Normal/Danger/Caution) 8dp 원형 점 (색상만)
버튼 텍스트 계속 진행 / 돌아가기 확인 / 다시하기

4. 시도했던 방법 (히스토리)

  • SelfTestResultGaugeBar: 가로 막대 + resultType별 2색 그라데이션 → Figma 반원형과 불일치
  • ResultGaugeGraph / SimpleFullCircleGauge: Canvas로 원형/반원 그리기 시도 → Figma의 250~500 스케일, 라벨, 인디케이터 구조와 미흡
  • SegmentedGaugeBar: 5등분 세그먼트 막대 (Figma 658:2777) → ST-10 디자인과 다른 컴포넌트
  • 점수 구간별 메시지, 습관 항목, 디스클레임, 버튼은 구현됐으나 Figma 레이아웃/비주얼과 정확히 맞지 않음

5. 필요한 결과

  • Figma ST-10 (node 317-3002)과 시각적으로 동일한 자가테스트 결과 화면
  • 반원형 게이지: 250~500 스케일, 5색 그라데이션 아크, 중앙 점수, 삼각형 인디케이터
  • 제목, 습관 Row(아이콘+상태), 디스클레임, 버튼 모두 Figma 스펙 반영
